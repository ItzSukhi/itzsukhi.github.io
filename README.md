Built using plain HTML/CSS/JS and ready to deploy on GitHub Pages.

## How to publish (recommended)

### Option A — Publish from root (recommended clean URL)
1. Create a **new public repository** named exactly: `itzsukhi.github.io`
2. Push these files to the repository root
3. GitHub Pages will serve your site at: `https://itzsukhi.github.io`

### Option B — Publish under a project path
1. Create repo named `portfolio` (or `sukhwinder-portfolio`)
2. Push files
3. Enable GitHub Pages in repo settings → branch `main` / `gh-pages`
4. Site will be served from: `https://itzsukhi.github.io/portfolio/`
